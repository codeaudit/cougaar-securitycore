Defines objects used for access control on top of the COUGAAR architecture. Classes in this package provide implementations of access control enforcement. The AgentTrustService is a non-privileged interface to the TrustAttributeService, which is a node-level service that maintains buckets of objects, which map to a collection of one or more TrustAttribute(s). The TrustSet contains a set of trust attributes which can be enumrated through or accessed directly by type.
