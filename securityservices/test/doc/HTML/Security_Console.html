Security Console Quick Start Guide

The M&R security console has been developed by the University of Memphis. This guide explains how to install the security console.

 

Installation

 

1.        First, install and configure the security services. Refer to the security services configuration guide.

2.        Check out the security console module from the CVS repository. The CVS repository is located at cvs.ultralog.net/cvs/security/umemphis.

3.        Copy the data directory located under the security console top directory to the $COUGAAR_INSTALL_PATH/configs/security/ directory.

4.        Copy the umemphis.jar jar file from the $Security_Console/lib directory to the $COUGAAR_INSTALL_PATH/lib directory.

5.    Sign the umemphis.jar file if it has not been signed previously. Sign the jar by typing the following commands:

cd $COUGAAR_INSTALL_PATH/lib directory
jarsigner -keystore \

${COUGAAR_SECURITY_SERVICES} \

/test/configs/sign-jars/signingCA_keystore \

-storepass keystore umemphis.jar privileged

Note: ${COUGAAR_SECURITY_SERVICES} is the directory where the security services module was checked out.

1.        To run the security console go to the following directory:

${COUGAAR_SECURITY_SERVICES}/test/bin/securityconsole

and type:

./seconLocal.sh or ./secon.sh
